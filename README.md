# A股指标检测实时看板

本项目把研究框架落地为一个本地 Web 看板：实时监控 A 股“顶部概率评分”、估值/ERP、债券压制、股债相对强弱、市场宽度、杠杆成交与趋势确认，并显示数据源可靠性。

## 快速启动

```powershell
pip install -e .[test]
uvicorn ashare_indicator_monitor.app:app --reload --host 127.0.0.1 --port 8000
```

然后打开：

```text
http://127.0.0.1:8000
```

## 数据源设计

- 行情与成交：东方财富公开行情接口（带本地缓存、超时与重试）。
- 指数估值：中证指数官网估值文件；历史分位使用乐咕乐股长序列，并与中证指数最新估值交叉校验。
- 国债收益率与中债国债财富指数：中国债券信息网。
- 两融：金十聚合的沪深两市两融历史序列；网页会明确标注“第三方聚合”，生产部署建议替换为上交所/深交所或 Tushare Pro 官方链路。
- 宽度：东方财富全 A 实时快照计算上涨家数、下跌家数、60 日涨幅为正占比、成交额等。

所有核心数据都带有 `as_of`、`fetched_at`、`status` 和风险提示；如果接口失败，会优先读取本地缓存并标注为缓存/过期，不会伪造实时数据。

## 指标评分说明

评分越高，顶部风险越高：

- 0-40：健康上涨
- 40-60：估值偏热
- 60-75：顶部预警
- 75-85：高危顶部区
- 85+：顶部确认概率高

详细研究摘要见 `docs/research_summary.md`。
